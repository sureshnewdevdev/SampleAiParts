Frequently Asked Questions

This FAQ addresses common queries about architecture, deployment, authentication, and integration. Use it as a quick reference while implementing solutions.

  • Which regions support Azure OpenAI? Check Azure documentation; only approved regions appear during resource creation.
  • How do I choose model versions? Pin to a version in your deployment and plan for regression testing when upgrading.
  • Can I restrict network access? Yes, use private endpoints, IP rules, and disable public network access.
  • How is data stored? Prompts and completions may be logged; opt-out options and regional data boundaries apply per policy.
  • What about quotas? Quotas are per model family and region; request increases for production loads.
